# Maruti Suzuki Financial Model

This project presents a financial model built in Excel for Maruti Suzuki India Ltd.

## Project Components

- Historical Financial Statement Analysis
- Ratio Analysis
- Financial Forecasting
- WACC Calculation
- Discounted Cash Flow (DCF) Valuation

## Purpose

The objective of this project is to understand corporate financial analysis and valuation techniques used in equity research and investment banking.

## Tools Used

- Microsoft Excel
- Financial Modeling
- Corporate Finance Concepts
